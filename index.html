Політика конфіденційності GoKart UA

Дата набрання чинності: 30.08.2024

Ця Політика конфіденційності описує, як мобільний додаток GoKart UA збирає, використовує та захищає ваші персональні дані відповідно до вимог Google Play та чинного законодавства.

1. Які дані ми збираємо

  • Особисті дані: ім’я, дата народження та номер телефону для створення та управління обліковим записом.
  • Доступ до камери: використовується виключно для сканування QR-кодів.
  • Інформація про покупки: дані з чеків для відображення у профілі та нарахування бонусів.
  • Push-сповіщення: ми надсилаємо сповіщення про акції, новини та персональні оновлення.
  • Аналітика: ми використовуємо сторонні сервіси для збору технічних даних (наприклад, час використання, кількість запусків, функціонал, яким користуються найчастіше).

2. Як ми використовуємо ваші дані

  • Створення та управління обліковим записом.
  • Сканування QR-кодів для роботи з клубними картками.
  • Відображення покупок у профілі та нарахування бонусів.
  • Інформування користувачів через push-сповіщення.
  • Аналітика для покращення функціональності та зручності додатку.

3. Захист даних

Ми застосовуємо сучасні методи захисту (шифрування, захищені сервери) для запобігання несанкціонованому доступу, зміні або видаленню ваших даних.

4. Сторонні сервіси

Наш додаток використовує такі сервіси:

  • Google Firebase – для зберігання даних та надсилання push-сповіщень (Firebase Cloud Messaging).
  • Google Analytics for Firebase – для збору анонімних даних про використання додатку з метою покращення сервісу.

Будь ласка, ознайомтесь з їх політикою конфіденційності:

  • Google Privacy Policy
  • Firebase Privacy and Security

5. Права користувачів

Ви маєте право на доступ, зміну або видалення своїх даних. Для цього зв’яжіться з нами за контактами нижче.

6. Відмова від збору даних

Ви можете відключити збір даних у налаштуваннях додатку або обмежити push-сповіщення у налаштуваннях вашого пристрою.

Ми можемо оновлювати цю Політику. Нова версія завжди буде доступна у додатку та за посиланням на цій сторінці.
