Політика конфіденційності GoKart UA

Дата набрання чинності: 30.08.2024

Ця Політика конфіденційності пояснює, як додаток "GoKart UA" збирає, використовує та захищає ваші особисті дані.

1. Які дані ми збираємо

  • Особисті дані: Ім’я, дата народження та номер телефону для створення облікового запису.
  • Камера: Доступ до камери для сканування QR-кодів.
  • Дані про покупки: Інформація про чеки для відображення у профілі та нарахування бонусів.
  • Push-сповіщення: Надсилання сповіщень про акції та новини.
  • Аналітика: Ми використовуємо сторонні сервіси (наприклад, Google Analytics) для аналізу використання додатку.

2. Як ми використовуємо ваші дані

  • Для створення та управління вашим обліковим записом.
  • Для сканування QR-кодів та надання функціоналу додатку.
  • Для нарахування бонусів та відображення покупок у профілі.
  • Для інформування про акції через push-сповіщення.
  • Для аналізу поведінки користувачів з метою покращення додатку.

3. Як ми захищаємо ваші дані

Ми використовуємо сучасні технології для захисту ваших даних від несанкціонованого доступу та втрати. Дані зберігаються на захищених серверах із застосуванням шифрування.

4. Сторонні сервіси

Ми використовуємо сторонні сервіси, такі як Google Analytics та Firebase, для аналітики та відправки сповіщень. Ці сервіси можуть збирати дані про ваше використання додатку. Ознайомтесь з їх політикою конфіденційності для детальної інформації.

5. Ваші права

Ви маєте право на доступ, виправлення або видалення ваших даних. Ви можете відмовитися від збору даних або видалити обліковий запис, зв’язавшись з нами.

6. Відмова від збору даних

Ви можете відключити збір даних у налаштуваннях додатку або відмовитися від push-сповіщень у налаштуваннях вашого пристрою.

7. Зміни в Політиці конфіденційності

Ми можемо періодично оновлювати цю Політику конфіденційності. Ми повідомимо вас про будь-які зміни, розмістивши оновлену версію в додатку.
